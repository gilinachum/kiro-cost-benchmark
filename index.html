Methodology

Repeated Trials

Each scenario runs 5× per tool. We report mean values with variance noted. Single runs prove nothing.

Automated Scoring

Code generation tasks use test suites (binary pass/fail). Analysis tasks use rubric-based LLM grading with inter-rater checks.

Blind Evaluation

Tool identity is stripped before quality assessment. Evaluators don't know which tool produced which output.

Same Prompt, Fresh Session

Verbatim prompts delivered in clean sessions. No context carryover, no MCP tools, no custom instructions.

Version Pinned

Tool versions, model versions, and dates recorded for every run. Fully reproducible.

Cost Normalized

All costs use raw API/pay-per-use pricing. No subscription amortization. Apples to apples.
